Useful information

  • Column "Files" : number of files of the given benchmark for which an intermediate representation is generated by the front-end.
  • Columns "Time" average wall time per analysed file.
  • (Rank and Termite) Columns "LP" : average size (lines, columns) of the Linear Programming instances.
  • C2FSM (Rank's frontend) is not robust enough to parse other benchmarks than WTC.
  • T2 is not part of the comparison since the C-front-end is not yet maintained.
  • We did not manage to launch TAN, we got many errors while processing C-files and the tool is not documented.
